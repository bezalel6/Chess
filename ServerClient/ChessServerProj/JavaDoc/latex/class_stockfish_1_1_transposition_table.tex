\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_stockfish_1_1_transposition_table}{Transposition\+Table}} is an array of Cluster, of size cluster\+Count. Each cluster consists of Cluster\+Size number of \mbox{\hyperlink{struct_stockfish_1_1_t_t_entry}{TTEntry}}. Each non-\/empty \mbox{\hyperlink{struct_stockfish_1_1_t_t_entry}{TTEntry}} contains information on exactly one position. The size of a Cluster should divide the size of a cache line for best performance, as the cacheline is prefetched when possible.
