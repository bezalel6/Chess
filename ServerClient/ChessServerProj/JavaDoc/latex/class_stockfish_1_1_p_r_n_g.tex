\doxysubsection{Detailed Description}
xorshift64star Pseudo-\/\+Random Number Generator This class is based on original code written and dedicated to the public domain by Sebastiano Vigna (2014). It has the following characteristics\+:


\begin{DoxyItemize}
\item Outputs 64-\/bit numbers
\item Passes Dieharder and Small\+Crush test batteries
\item Does not require warm-\/up, no zeroland to escape
\item Internal state is a single 64-\/bit integer
\item Period is 2$^\wedge$64 -\/ 1
\item Speed\+: 1.\+60 ns/call (Core i7 @3.\+40\+GHz)
\end{DoxyItemize}

For further analysis see \href{http://vigna.di.unimi.it/ftp/papers/xorshift.pdf}{\texttt{ http\+://vigna.\+di.\+unimi.\+it/ftp/papers/xorshift.\+pdf}}
