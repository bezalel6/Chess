\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_stockfish_1_1_move_picker}{Move\+Picker}} class is used to pick one pseudo-\/legal move at a time from the current position. The most important method is \mbox{\hyperlink{class_stockfish_1_1_move_picker_a11368e6ef0025e37d64e62f77b88d17a}{next\+\_\+move()}}, which returns a new pseudo-\/legal move each time it is called, until there are no moves left, when MOVE\+\_\+\+NONE is returned. In order to improve the efficiency of the alpha-\/beta algorithm, \mbox{\hyperlink{class_stockfish_1_1_move_picker}{Move\+Picker}} attempts to return the moves which are most likely to get a cut-\/off first.
