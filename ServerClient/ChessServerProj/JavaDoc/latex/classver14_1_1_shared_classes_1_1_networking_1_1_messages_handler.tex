\doxysubsection{Detailed Description}
Messages handler -\/ handles all types of messages by using a hash map to make all routing as fast as possible. when a request is sent to the server, a callback is passed with it. when a response is received, that callback is called with the response message. ~\newline
 if a message that isn\textquotesingle{}t a response is received, the handling is differed to the default callbacks map. which is where most of the implementation of this abstract class comes in. the individual message type handling
