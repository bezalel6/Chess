\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_stockfish_1_1_pawns_1_1_entry}{Pawns\+::\+Entry}} contains various information about a pawn structure. A lookup to the pawn hash table (performed by calling the probe function) returns a pointer to an \mbox{\hyperlink{struct_stockfish_1_1_pawns_1_1_entry}{Entry}} object.
