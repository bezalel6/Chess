\doxysubsection{Detailed Description}
\mbox{\hyperlink{enumver14_1_1_shared_classes_1_1_game_1_1_moves_1_1_direction}{Direction}} -\/ represents a moving direction on a board. sort of like a vector. has a {\ttfamily \mbox{\hyperlink{enumver14_1_1_shared_classes_1_1_game_1_1_moves_1_1_direction_a68e714d3c283f95e5ca2b2c104e76ad9}{offset}}} that is added to a certain location or bitboard, in order to achieve movement in that direction. the general direction map looks like this\+: \tabulinesep=1mm
