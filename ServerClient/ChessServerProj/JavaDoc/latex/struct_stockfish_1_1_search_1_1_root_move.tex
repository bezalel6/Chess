\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_stockfish_1_1_search_1_1_root_move}{Root\+Move}} struct is used for moves at the root of the tree. For each root move we store a score and a PV (really a refutation in the case of moves which fail low). Score is normally set at -\/VALUE\+\_\+\+INFINITE for all non-\/pv moves.
