\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_stockfish_1_1_search_1_1_stack}{Stack}} struct keeps track of the information we need to remember from nodes shallower and deeper in the tree during the search. Each search thread has its own array of \mbox{\hyperlink{struct_stockfish_1_1_search_1_1_stack}{Stack}} objects, indexed by the current ply.
